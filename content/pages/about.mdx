## What this blog is now

This is a tips-and-tricks hub for people building with MDX and Next.js. Expect
short, opinionated posts that focus on the moments that actually save time: file
structure, content automation, component-driven writing, and shipping a
polished template without the drama.

## Where to go next

- Read the latest articles at [/blog](/blog).
- Find code samples and small apps at [/code](/code).
- Clone the latest Next.js 16 template from
  [github.com/owolfdev/mdx-nextjs-latest](https://github.com/owolfdev/mdx-nextjs-latest).

## The template, in one sentence

It is a modern MDX-first Next.js 16 starter with sensible defaults, a clean
design system, and just enough structure to publish quickly without feeling
boxed in.
